# Project — AI-Powered Workspace Landing Page

A clean, modern landing page built with HTML and Tailwind CSS, designed to showcase an AI-powered productivity workspace.

## What This Project Is

This project is a modern digital workspace platform that helps teams to write, plan and collaborate effectively with the help of AI tools.


## What's Inside

The page primarily focuses on the hero section.

**The Hero Section** is the big, bold centerpiece of the page. It includes sticky navigation, headline structure that features a headline ("Write, plan, share. With AI at your side."), call-to-action buttons, floating information cards, positioned dashboard preview, and a fixed chat widget. The design demonstrates the use of Tailwind CSS positioning utilities(relative, absolte, stickty, fixed) along with spacing, layot structuring , and UI components.  

## Project Structure

The project is intentionally minimal:

/
└── index1.html    # The entire page — markup, layout, and CDN links all in one file

Everything lives in a single HTML file, which makes it easy to share, fork, or embed as a starting point.
